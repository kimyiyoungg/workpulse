# WorkPulse

**업무 효율성을 높여주는 사내 정보 공유 플랫폼**

## 🖥️ 프로젝트 소개

**WorkPulse**는 조직 내 원활한 소통과 정보 공유를 위한 사내 정보 실시간 공유 플랫폼입니다.  
팀원 간의 협업과 커뮤니케이션을 효율적으로 지원하는 다양한 기능들을 제공합니다.

---

## 🕰️ 개발 기간

2025.01.01 ~ 2025.04.14

---

## 🧑‍🤝‍🧑 멤버 구성

- 김이영  
- 박예린

---

## ⚙️ 개발 환경

- **Language**: TypeScript  
- **IDE**: Visual Studio Code  
- **Framework**: React  
- **Database**: Firebase (v14.1.0)

---

## 📌 주요 기능

### 🔐 회원 기능
- 회원가입 / 로그인
- Firebase Authentication 사용
- Google OAuth2 로그인 연동

### 📝 업무 게시판
- 게시글 작성 / 읽기 / 수정 / 삭제 (CRUD)
- 이미지 등록 가능

### 🎉 경조사 알림
- 경조사 게시글 CRUD
- 모달 창 활용 UI

### 🙋‍♂️ 프로필 관리
- 프로필 사진 변경
- 부서 및 한줄소개 수정 기능

### 🗒️ 개인 메모
- 개인화된 메모 CRUD 기능 제공

### 💬 단체 채팅
- 실시간 단체 채팅 기능 구현
